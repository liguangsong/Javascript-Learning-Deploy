函数的声明

可以通过函数声明语句来定义一个函数.函数声明语句以关键字 function 开始,其后跟有 函数名 参数列表 和 函数体代码块.

声明函数的格式:

    function 函数名(参数, 参数, ...){
        代码块
    }

例如:

    //  函数 sum 的声明
    function sum(a, b){
        return a+b ;
    }

其中 sum 为函数名, a 和 b 为参数名, 函数体中对两个参数进行了加运算, 并通过 return 语句返回结果.

将代码写成函数的形式之后,函数内的代码就只能通过调用该函数的方式来执行.
